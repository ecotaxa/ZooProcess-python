About ZooProcess

What is ZooProcess?

ZooProcess is a powerful tool for processing and analyzing plankton images. It provides a comprehensive suite of tools for image processing, data management, and analysis of plankton samples.

Features

  • Image processing and analysis
  • Data management and organization
  • Integration with EcoTaxa for taxonomic classification
  • Automated processing workflows
  • User-friendly web interface
